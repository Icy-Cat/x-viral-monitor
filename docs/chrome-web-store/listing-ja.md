# Chrome Web Store - 商品詳細（日本語）

## 簡潔な説明（132 文字以内）

```
バズ投稿の検出・長尺画像の閲覧・Markdown コピー・サポーター星図・Grok 返信補助 — すべてローカル処理、データ収集なし。
```

## 詳細な説明

```
X バイラルモニターは、X（Twitter）のタイムラインをコンテンツ発見・閲覧・エンゲージメントのコントロールパネルに変えます。すべての処理はお使いのブラウザ内で完結し、データはどこにも送信されません。

⸻ 流速バッジ ⸻
• すべてのツイートに「閲覧数/時」のバッジをリアルタイム表示
• 3 段階：🌱 通常 · 🚀 上昇 · 🔥 バズ
• ポップアップから閾値をカスタマイズ可能
• ホバーで詳細指標を表示：閲覧、いいね、リポスト、返信、ブックマーク、バズ指数

⸻ ページ内ホットランキング ⸻
• ドラッグ可能なフローティングパネルで、現在表示中の急上昇ツイートを順位付け
• 行をクリックするとそのツイートへジャンプ、戻るボタンで元の位置へ復帰
• カラムの並び替え対応

⸻ 強化版フォトビューア ⸻
• X のライトボックス上でホイール拡大縮小、ドラッグ移動、ダブルクリック切替、+/- /0 ホットキー対応
• NEW：長尺画像リーディングモード — 縦長スクリーンショットを 900px の閲覧幅で縦スクロール表示し、画質を 4096px に自動アップグレード（潰れません）

⸻ Markdown としてコピー ⸻
• X のシェアメニューに「Markdown としてコピー」を追加
• 著者、日付、リンク、本文を整形済みの Markdown でクリップボードへ

⸻ Thank-You サポーター星図 ⸻
• 任意のツイートのシェアメニューをワンクリックすると、リポスト・引用したユーザー全員を可視化
• 検索可能なサポーター一覧、リポスト/引用フィルター、引用リバー（タイムライン表示）

⸻ Grok 返信アシスタント ⸻
• X 内蔵の Grok エンドポイント（X ウェブが実際に利用しているのと同じもの）を呼び出して返信案を生成
• プロンプトテンプレートを編集可、一時チャットモード（Grok 履歴を汚さない）対応

⸻ 仕組み ⸻
本拡張機能は X が既に発行している GraphQL レスポンスと同一オリジンのエンドポイントにフックします — サードパーティサーバーへの通信、解析、トラッキングは一切ありません。すべての処理はブラウザ内で完結します。

⸻ プライバシー ⸻
✓ 閲覧中の x.com / pro.x.com のページデータのみを読み取り
✓ 第三者ネットワークリクエストなし、解析なし、トラッキングピクセルなし
✓ ローカル保存はユーザー設定とサポーター星図キャッシュのみ（chrome.storage）
✓ 完全オープンソース：https://github.com/Icy-Cat/x-viral-monitor

⸻ 対応環境 ⸻
• x.com と pro.x.com（TweetDeck）対応
• UI 言語：English / 中文 / 日本語
```
