# Chrome Web Store - 商品详情（中文 - 中国）

## 简短说明（132 字符以内）

```
推文流速徽章、长图阅读、Markdown 复制、转发星图、Grok 回复辅助 — 全部本地处理，零数据收集。
```

## 详细说明

```
X 爆款监测把 X（Twitter）时间线变成内容捕捉、阅读和互动的控制面板——所有处理都在你的浏览器本地完成，不上传任何数据。

⸻ 流速徽章 ⸻
• 在每条推文上实时显示"浏览量/小时"徽章
• 三档：🌱 普通 · 🚀 上升 · 🔥 爆款
• 通过 popup 自定义阈值
• 鼠标悬停查看详细指标：浏览、点赞、转发、回复、收藏、爆帖指数

⸻ 当前页面热榜 ⸻
• 可拖动的浮动面板，列出当前可见的最快速增长推文
• 点击任意一行跳转到该推文，返回按钮回到原位置
• 列可拖动重排

⸻ 增强图片查看器 ⸻
• 在 X 灯箱里支持滚轮缩放、拖拽平移、双击切换、+/- /0 快捷键
• 新增：长图阅读模式 — 长截图以 900px 阅读宽度纵向滚动展示，并自动升级到 4096 像素画质，不再被压缩

⸻ Copy as Markdown ⸻
• 在 X 的分享菜单中添加"复制为 Markdown"
• 一键复制作者、日期、链接和正文为干净的 Markdown 格式

⸻ Thank-You 转发星图 ⸻
• 在任意推文的分享菜单点一下，可视化所有转发或引用过这条推文的人
• 可搜索的支持者列表、转发/引用筛选、引用河流时间线

⸻ Grok 回复助手 ⸻
• 调用 X 内置的 Grok 端点（与 X 网页本身使用的同一端点）生成回复草稿
• 提示词模板可编辑、可选"临时聊天"模式（不污染 Grok 历史）

⸻ 工作原理 ⸻
扩展挂接 X 自身已发起的 GraphQL 响应与同源接口——不联系任何第三方服务器、无统计、无追踪。所有处理在你浏览器本地完成。

⸻ 隐私 ⸻
✓ 仅读取你正在访问的 x.com / pro.x.com 页面数据
✓ 不发起任何第三方网络请求、无统计、无追踪像素
✓ 仅本地存储你的设置 + 一份本地星图缓存（chrome.storage）
✓ 完全开源：https://github.com/Icy-Cat/x-viral-monitor

⸻ 兼容性 ⸻
• 支持 x.com 和 pro.x.com（TweetDeck）
• 界面语言：English / 中文 / 日本語
```
