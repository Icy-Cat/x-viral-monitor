# Privacy Policy — X Viral Monitor

**Last updated:** April 2026

## Data Collection
X Viral Monitor does **not** collect, store, or transmit any personal data or browsing data to external servers.

## What the extension accesses
- **X GraphQL API responses**: The extension reads tweet metrics (view counts, likes, retweets, replies, bookmarks, post timestamps) from API responses that X's website already loads in your browser. It does not make any additional network requests.
- **chrome.storage.sync**: Used solely to store your custom velocity threshold settings. This data syncs across your Chrome instances via your Google account and contains only two numbers (trending threshold and viral threshold).

## What the extension does NOT do
- Does not collect any personal information
- Does not track browsing history
- Does not make any external network requests
- Does not inject ads or analytics
- Does not access or store login credentials
- Does not share any data with third parties
